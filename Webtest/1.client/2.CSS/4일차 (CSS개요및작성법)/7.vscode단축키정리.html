Visual studio code에서 자주 사용되는 단축키

  1. html문서 템플릿 작성법 : html파일 작성 후 ! enter or html:5 enter
  2. 태그 작성하는 방법 : 태그이름 enter -> 자동 완성 기능 ex) div enter
  3. 태그를 여러 개 생성하는 방법 : 태그명*n(반복할 수) -> ex) div*5
  4. 태그를 중첩 시키는 방법 : 태그명>태그명2>태그명3 -> ex) div>ul>ol*3 (*3은 응용)
        1. 인접한 태그 생성하는 방법 : 태그명+태그명2+태그명3 ->ex) div+h1+h2+h3

        2. 아이디 선택자 설정 : 태그명#아이디명(임의 지정) -> ex) div#name
        3. 클래스 선택자 설정 : 태그명.클래스명(임의 지정) ->ex) div.name
        4. 태그{입력하고 싶은 텍스트} 입력 후 enter or tab -> ex) p{안녕하세요.}

          안녕하세요.

        5. 숫자 삽입(숫자 1씩 자동 증가) 방법: 아이디, 클래스, 텍스트에 자주 사용 -> ex) div#name$*n(반복할 숫자)
        6. ^입력 시 상위 계층으로 이동(상위와 동일한 위치로 간다.) (중첩된 상황에서 상위 계층으로 이동할 때, 여러번 사용가능) ->ex) div>ul>li^p

        7. ^^입력 시 상상위 계층으로 이동 -> 형식) div>ul>li^^p

        8. (태그...) 그룹으로 묶는 경우 -> ex) div>(ul>li)+(ol>li) 입력 후 enter
        9. 자동완성기능 div#myid.classA.classB 입력 후 enter 동시에 div태그에 id와 class선택자를 만드는 방법(div#myid.classA 같은 형식 가장 많이사용)
          (class도 누적 적용 가능)
